# baziteller
八字是中国古代宇宙观、生命观与阴阳五行思想的结晶，具有深厚的人文价值。应以理性态度看待：命理不是替代医学、经济学或心理学的决策工具，而是一种文化自省与自我认知的辅助手段。
功能简介
八字测算 Skill 基于中国传统的四柱命理学，根据用户提供的公历出生日期、时间和性别，自动推算以下内容：

四柱八字：年柱、月柱、日柱、时柱（天干+地支）

纳音五行：年柱对应的纳音（如“海中金”、“炉中火”）

五行分布：四柱天干和地支各自对应的五行属性

十神关系：以日干（日元）为中心，分析其他天干和地支主气十神（比肩、劫财、食神、伤官、正财、偏财、正官、七杀、正印、偏印）

大运排盘：根据性别和年干阴阳，计算大运的顺逆方向及前10步大运干支

流年干支：可查询任意年份的流年干支（默认当前年份）

该 Skill 严格遵循传统命理规则：年柱以立春为界，月柱依节气划分，日柱采用公式计算，时柱按五鼠遁，大运按阳年男/阴年女顺排、反之逆排。
